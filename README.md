## ✨ Features

- **Loading Screen** — animated logo with progress bar
- **Scroll Progress Bar** — gradient bar at the top of the page
- **Active Navbar** — highlights current section while scrolling, collapses to hamburger on mobile
- **Dark / Light Mode Toggle** — persists across sessions via `localStorage`
- **Typed.js Animation** — rotating role titles in the hero section
- **AOS Animations** — smooth scroll-reveal on every section
- **Animated Skill Bars** — triggered when the section enters the viewport
- **Counter Animation** — stat numbers count up on scroll
- **Certificate Lightbox** — click any certificate to open a full-screen modal
- **Contact Form** — client-side with success feedback
- **Resume Section** — download and view PDF
- **Back to Top Button**
- **Glassmorphism Cards** throughout
- **Rotating gradient border** on profile image
- **Fully Responsive** — desktop, laptop, tablet, mobile

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/khandelwalmahak30-prog/Mahak-Portfolio.git

# Open in browser
cd Mahak-Portfolio
start index.html
```

No build steps. No npm install. No dependencies to manage.

---

## 🌐 Deploy on GitHub Pages

1. Push the project to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your portfolio will be live at `https://khandelwalmahak30-prog.github.io/Mahak-Portfolio/`
